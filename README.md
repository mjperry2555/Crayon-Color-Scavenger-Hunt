 Crayon Scavenger Hunt 

Interactive color-matching game for Pythonista on iOS using perceptual color science.

Pick a photo, tap any object, and discover the closest Crayola color.

Uses RGB → CIE Lab conversion and CIEDE2000 color difference matching.

Tracks discovered colors and saves progress between sessions.

Designed for natural-light photos and real-world color exploration.

Features a large Crayola-inspired color palette and touch-based sampling.

Built with Pythonista `ui`, `photos`, and Pillow (PIL).

Progress can be reset at any time from the in-app Reset button.

A fun way to learn colors, collect discoveries, and explore your surroundings.
